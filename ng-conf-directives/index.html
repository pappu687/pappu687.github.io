Let's get started

Steps of writing a directive

1 Define Angular Module 

								angular.module("myDirectives",[]);
2 Load module in Application 

								angular.module("myApp",["myDirectives"]);

Steps of writing a directive (contd.)


3 Register Directive 

			angular.module("myDirectives",[])
			.directive("userProfile",[function(){
			   return {
			     ...
			   }
			}]);
4 Use your awesome directive in HTML

					<user-profile data="userProfileData"></user-profile>

What do I use, Controller or Link in Directives ?


Before Compilation => Controller

After Compilation => Link

Directive Best Practices


1 Prefer tags and attributes instead of comments and class

2 Use dash delimitted names. e.g. <user-profile>

3 Return definition object instead of function

4 Prefix your directives. Just don't use ng- prefix :)

5 Use templateUrl if your template is not too tiny.

6 Use Controller when you want to expose API to other directives

7 Directives should clean up after themselves. You can use element.on('$destroy', ...) or scope.$on('$destroy', ...) to run a clean-up function when the directive is remove
